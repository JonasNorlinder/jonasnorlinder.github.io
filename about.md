Been hacking on computers for as long as I can remember, but I have been told that already at the age of three my parents could find me looking for fuse boxes because I wanted to "see how it works". Luckily, I survived that phase, and I am currently pursuing a PhD in execution environments for high-level programming languages. Most of this work has so far involved implementing and evaluating different approaches for reducing the memory footprint and improving the object layout in OpenJDK/[ZGC](https://wiki.openjdk.java.net/display/zgc/Main).
